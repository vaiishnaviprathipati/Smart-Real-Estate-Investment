Smart Real Estate Investment Risk Analyzer is a data-driven application designed to help investors evaluate real estate properties and make informed investment decisions. The system analyzes key property factors, calculates risk levels, and provides insights that support smarter investment planning.

Features
Property investment analysis
Risk assessment and scoring
Investment decision support
Data visualization and reporting
User-friendly interface
Real estate data processing
Project Objectives
Analyze real estate investment opportunities
Identify potential investment risks
Provide data-driven recommendations
Improve investment decision-making through analytics

python main.py
